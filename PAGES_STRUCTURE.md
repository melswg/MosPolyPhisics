# Структура страниц проекта «МосПолиФизикс»

Этот файл описывает все страницы сайта, их функционал и необходимые данные из API.

## Глобальные элементы (на всех страницах)
*   **Header:** Логотип, навигация (О нас, Новости, Проекты, Тесты, Личный кабинет).
*   **Footer:** Ссылки на соцсети, политика конфиденциальности, контакты.

---

## Детальное описание страниц

### 1. Главная (index.html)
*   **Цель:** Хаб сайта.
*   **Контент:** Блок «Цитата дня» (динамический), описание проекта, 4 карточки-перехода (О нас, Новости, Проекты, Тесты).
*   **API:** `GET /api/quote` — для получения цитаты при загрузке.

### 2. О нас (pages/about.html)
*   **Цель:** Информирование.
*   **Контент:** Текст о проекте, рубрики (видео, комиксы и т.д.), блок «Команда проекта».
*   **API:** Пока статика, данных не требует.

### 3. Новости (pages/news.html)
*   **Цель:** Информирование о событиях.
*   **Контент:** Список карточек новостей (заголовок, дата, текст, картинка).
*   **API:** `GET /api/news` — получение всех новостей.

### 4. Проекты (pages/project.html)
*   **Цель:** Навигация по проектным блокам.
*   **Контент:** Карточки: Видео, Календарь, Визуальная новелла.

### 5. Видео (pages/video.html)
*   **Цель:** Просмотр контента.
*   **Контент:** Сетка из видео (iframe или ссылки).
*   **API:** (Опционально) `GET /api/videos` — список ссылок.

### 6. Календарь (pages/calendar.html)
*   **Цель:** Расписание событий.
*   **Контент:** Интерактивный календарь с выделенными датами.
*   **API:** `GET /api/calendar` — список событий.

### 7. Визуальная новелла (pages/novel.html)
*   **Цель:** Медиа-контент.
*   **Контент:** Кнопка «Скачать», Скриншоты (слайдер), Дневник разработки (список), Лор игры.
*   **API:** (Опционально) `GET /api/novel/updates`.

### 8. Тесты (pages/tests.html)
*   **Цель:** Интерактив.
*   **Контент:** 4 карточки (Тест по физике, Цитаты, Задания ЕГЭ, Развлекательные).
*   **API:** `GET /api/tests` — список доступных тестов.

### 9. Шаблон теста (pages/test_template.html)
*   **Цель:** Прохождение теста.
*   **Контент:** Вопрос, варианты ответов, кнопки навигации (назад/вперед).
*   **API:** `GET /api/test/{test_id}` и `POST /api/test/submit`.

### 10. Личный кабинет (pages/account.html)
*   **Цель:** Управление профилем.
*   **Контент:** Аватар, имя, почта, кнопка «Выйти».
*   **API:** `GET /api/user/me` (данные профиля).

### 11. Вход / Регистрация (pages/login.html, pages/register.html)
*   **Цель:** Авторизация.
*   **Контент:** Формы (имя, почта, пароль).
*   **API:** `POST /api/register`, `POST /api/login`.
