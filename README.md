# МосПолиФизикс 

Образовательный веб-ресурс для студентов и абитуриентов, посвященный физике. Проект разрабатывается в рамках университетской программы.

## Функционал прототипа
- **ХАБ (главная страница):** навигация, маскот "Симплик", динамический блок «Цитата дня».
- **Информационный блок:** разделы «О нас» и «Новости».
- **Интерактив:** подготовка к ЕГЭ, тесты и цитаты великих физиков.
- **Проектный блок:** видеоматериалы, календарь событий и визуальная новелла.
- **Личный кабинет:** регистрация и вход пользователей.

## Стек технологий
- **Frontend:** HTML5, CSS3 (адаптивный дизайн, темная/светлая темы), JavaScript.
- **Backend:** Python 3, FastAPI.
- **Database:** SQLite.
