Медиафайлы для МосПолиФизикс (макеты Figma)
==========================================

Корень: frontend/media/  (в HTML с главной — media/..., со страниц pages/ — ../media/...)

Рекомендуемая структура и имена (после экспорта из Figma подложите файлы сюда — сайт подхватит их по этим путям):

icons/
  user-circle.svg          — иконка профиля в шапке (уже есть заглушка; можно заменить экспортом)
  nav-about.svg …          — опционально: свои иконки навигации вместо Font Awesome

backgrounds/
  pattern-physics.svg      — лёгкий паттерн для фона секций (уже есть; можно заменить полноценным из макета)
  hero-formulas.png        — опционально: текстура «доска с формулами» для hero (если добавите — раскомментируйте в figma.css)

news/
  news-<id>-cover.jpg      — обложки к новостям, например news-1-cover.jpg (подключение в разметке вручную или позже из API)

videos/
  thumb-<slug>.jpg         — превью для сетки видео, например thumb-demo.jpg

novel/
  sprite-hero.png          — спрайт персонажа
  screenshot-01.jpg …      — скриншоты для карусели

avatars/
  default-avatar.png       — аватар по умолчанию в личном кабинете

Корень media/
  logo.svg                 — логотип текстовый в шапке/футере
  mas.svg                  — маскот на главной

Подсказка: экспорт из Figma — SVG для иконок и логотипа, JPG/WebP 1200px для фото, имена в kebab-case.
